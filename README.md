# css-color-name-classes
CSS background and text color classes by color name.

140 color names included that are supported by all modern browsers.

## Example

```css
/* Background class */
.DarkGoldenRodBg{
  background-color: DarkGoldenRod;
}

/* Text class */
.DarkGoldenRodTxt{
  color: DarkGoldenRod;
}
```
